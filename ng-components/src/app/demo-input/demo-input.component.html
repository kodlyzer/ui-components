Input Component

Test Inputs (JSON DATA)

  • Description

    • Input Fields allow users to enter inputs into a UI. They typically appear in forms and dialogs..

  • Supported <input> types
    • Text
    • Color
    • Date
    • Month
    • Url
    • Time
  • Configuration Options

    • Use this Selector in your component to include Badge Component.

              <kdi-app-input [type]="input.type" [value]="input.value" [size]="input.size" [placeholder]="input.placeholder" [label]="input.label">
          </kdi-app-input>
  • How to use ?

    • Follow the following format to use it.

              <kdi-app-input [type]="input.type" [value]="input.value" [size]="input.size" [placeholder]="input.placeholder" [label]="input.label">
              </kdi-app-input>   

            We can used like below:
              <kdi-app-input [type]="color" [value]="#f00" [size]="base" [placeholder]="color code" [label]="Color Code">
              </kdi-app-input>

    For the list of "type"s available refer to KDI design system.
