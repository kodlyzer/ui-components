Badge Component

Test Inputs (JSON DATA)

  • Description

    • The checkbox is shown as a square box that is ticked (checked) when activated.Checkboxes are used to let a user select one or more options of a limited number of choices. Checkboxes are for selecting one or several options in a list, while radios are for selecting one option from many

  • Configuration Options

    • Use this Selector in your component to include Checkbox Component.

              <kdi-checkbox [checked]="demoCheckbox1.checked" [label]="demoCheckbox1.label" [disabled]="demoCheckbox1.disabled">
          </kdi-checkbox>
  • How to use ?

    • Follow the following format to use it.

                   demoBadge= {  color:"name of the color (from the list of colors provided by KDI)",  
                label:"name of the label you want to display "   } ;                         
             Here we have used :
               demoBadge1= {  color: 'primary',  label:'Primary' } ;
               demoBadge2= { color:'success', label:'Success'  } ;
               and so on.    
               
               Note: If you don't provide the "color" value, by default it will give the "foam" as color.

    The list of colors present now are primary, secondary, success, danger, foam, navy and warning.
