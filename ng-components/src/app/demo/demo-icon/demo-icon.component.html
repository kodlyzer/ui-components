Icon Component

Test Inputs (JSON DATA)

  • Description

    • An icon can be used to symbolize a command, file, device, or directory. System icons are also used to represent common actions like trash, print, and save, and are commonly found in app bars, toolbars, buttons, and lists. In order to use one from the set of icons provided by KDI follow the guidelines given below.

  • Configuration Options

    • Use this Selector in your component to include Icon Component.

              <kdi-icon [icon]="iconName">
          </kdi-icon>
  • How to use ?

    • Follow the following format to use it.

               iconName="name of the icon you want to display"; 
          
          
         Here we have used :
           iconName1="box" ;
           iconName2="search";
           iconName3="bookmark"
           iconName4="angle-left"

    The list of the icons are present in our assets/img/icons folder. Just assign the icon name to the "iconName" variable. (.svg extension after the icon name is not required)
