Change Notification type : 

 Syntax: <kdi-notification [is{{status}}]= boolean >
  • Description

    • A notification is a message that application displays in your app's UI to provide the user with reminders, communication from other people, or other timely information from your app

  • Configuration Options

    • Use this Selector in your component to include Notification Component.

                <kdi-notification [option]= boolean >
                <kdi-notification-header >
                  //notification header
                </kdi-notification-header>

                <kdi-notification-content >
                  //conent for notifcation
                </kdi-notification-content>
            </kdi-notification>
  • How to use ?

    • Follow the following option you can use it.

    • isNeutral
    • isSuccess
    • isDanger
    • isWarning
    • isCritical 
                <kdi-notification [option]= boolean > for notification component

            <kdi-notification-header > for giving header to your notification

            <kdi-notification-content > for content for your notifcation it can include image paragraph tag.
