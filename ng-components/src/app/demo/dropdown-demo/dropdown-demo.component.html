  • Configuration Options

    • Use this Selector in your component to include DropDown Component.

      <kdi-dropdown 
    [dropdown]="dropdown" 
    (getClickPath)="handler($event)">
  </kdi-dropdown>
  • How to use?

    • Use this Format for specifying your content for your dropdwon component and pass is to the propety in the secector tag.

        dropdown = {
      title: 'Courses',

  • How to listen to select item from drop down?

    • Use this EventListener in your selector tag to listen to the selected path and you have to create a event handler function in your component. 

        Add this in your selecor tag (getClickPath)="handler($event)"
    
    handler(path: string) {
      this.path = path;
